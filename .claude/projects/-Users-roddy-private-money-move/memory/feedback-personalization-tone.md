레슨 본문에서 "내 숫자"(`my:` 토큰, [[differentiation-my-numbers]])를 설명할 때 사용자를 2인칭으로 직접 부르는 문구("너로 치면", "너라면")는 쓰지 않는다. "사용자님은", "사용자님 기준으로는" 같은 존댓말 어감으로 통일한다.

**Why:** "너"는 반말이라 건방지게 느껴진다는 피드백(2026-07-06). 기존 레슨들은 애초에 2인칭 지칭 자체를 거의 안 쓰고 "-예요/-어요" 서술형으로만 쓰는데, 개인화 문구를 추가하면서 처음으로 직접 지칭이 필요해졌고 거기서 반말이 나왔다.

**How to apply:** 앞으로 어떤 레슨에든 개인화 문구(`my:` 토큰 주변 설명)를 추가할 때, 사용자를 직접 부를 일이 있으면 항상 "사용자님"을 쓴다. 가능하면 지칭 자체를 생략하고 서술형으로 자연스럽게 풀 수 있는지 먼저 고려하되(기존 톤과 더 잘 맞음), 지칭이 필요하면 "사용자님"이 기본값이다.
